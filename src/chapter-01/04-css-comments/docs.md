# CSS Comments

Css solo soporta los comentarios de bloque.

```css
/* HERE GOES A COMMENT */
.red {
  color: red;
}
```

Se pueden comentar secciones enteras de esta manera

```css
/*
.red {
  color: red;
}

.blue {
  color: blue;
} */
```

Los comentarios en línea de javascript no funcionan

```
.red {
  color: red; // comentario en línea no funciona
}
```
